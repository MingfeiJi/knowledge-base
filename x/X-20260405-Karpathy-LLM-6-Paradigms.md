# X Thread: Karpathy 的 2025 LLM 六大范式转变

## Thread

**1/8**
Karpathy 总结了 2025 年 LLM 领域的六大范式转变，每一个都在重新定义我们对 AI 的认知。

一个 thread 帮你快速理解这一年发生了什么

**2/8**
1️⃣ RLVR (强化学习从可验证奖励)

LLM 训练从三阶段变成四阶段：
预训练 → SFT → RLHF → RLVR

关键突破：模型学会自发"推理"
里程碑：OpenAI o1 演示，o3 发布

**3/8**
2️⃣ "幽灵 vs 动物"

LLM 不是"进化中的动物"，而是"被召唤的幽灵"

它可以同时是：
• 博学的专家
• 容易被欺骗的小学生

智能是"参差不齐"的，不是线性进化

**4/8**
3️⃣ Cursor 揭示新架构层

LLM 应用的核心能力：
• 上下文工程
• 复杂 LLM 调用编排
• 应用特定 GUI
• "自主程度滑块"

预测：LLM 应用将成为垂直专家

**5/8**
4️⃣ Claude Code - 本地 AI Agent

首次真正实用的 LLM Agent

关键特性：
• 运行在你的电脑上
• 访问你的环境和数据
• 低延迟交互

像一个"住在你电脑里的小精灵"

**6/8**
5️⃣ Vibe Coding (氛围编程)

AI 跨越了"用英语写程序"的能力阈值

影响：
• 编程不再需要专业训练
• 代码变得一次性、可丢弃
• 普通人获得软件开发能力

**7/8**
6️⃣ LLM GUI 萌芽

当前的"聊天"像 1980 年代的命令行

人类倾向视觉/空间消费
文本只是过渡形态

需要：整合文本生成、图像生成、世界知识的 GUI 层

**8/8**
Karpathy 的关键判断：

"我们对现有能力的利用率可能只有 10%"

整个领域 "conceptually wide open"

2025 是范式转变年
2026 将是落地应用年
