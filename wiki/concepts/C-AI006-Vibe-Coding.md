# Vibe Coding (氛围编程)

## 定义

使用自然语言（主要是英语）描述需求，由 AI 自动生成程序代码的编程范式。

## 详细解释

Vibe Coding 代表 AI 跨越了仅用英语构建程序的能力阈值。这一概念由 Andrej Karpathy 在 2025 年提出，标志着编程门槛的根本性降低。

核心理念：
- 编程不再局限于训练有素的专业人士
- 代码变得"免费、短暂、可塑、一次性使用后可丢弃"
- 专业人士能编写更多原本不会存在的软件

Vibe Coding 符合 Karpathy 关于 LLM "赋予普通人权力"的理论。它将软件开发从技能密集型活动转变为意图表达活动。

## 关键特征

- 使用自然语言描述需求
- 降低编程门槛
- 代码即时生成、即时使用
- 强调快速迭代而非代码质量

## 典型例子

- 非程序员快速原型验证想法
- 专业开发者快速生成样板代码
- 一次性数据处理脚本
- 快速 UI 原型
