# Karpathy 的 2025 LLM 年度回顾

## TL;DR

Karpathy 总结 2025 年 LLM 领域六大范式转变：RLVR 训练方法、"幽灵 vs 动物"认知框架、Cursor 应用层、Claude Code 本地 Agent、Vibe Coding 编程范式、LLM GUI 萌芽。

## 核心结论

- RLVR 成为训练第四阶段，使模型自发展现推理能力
- LLM 是"被召唤的幽灵"而非"进化的动物"，呈现"参差不齐"的智能
- Claude Code 首次展现 LLM Agent 在本地计算机上的实际应用
- Vibe Coding 让非专业人士也能用英语构建程序
- 基准测试容易被 RLVR 针对性优化，存在过度拟合风险

## 关键证据

- OpenAI o1 首次演示 RLVR，o3 发布成为转折点
- Cursor 揭示 LLM 应用的新架构层次
- LLM 现有能力利用率不足 10%
