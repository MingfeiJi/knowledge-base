# 术语表 (Glossary)

巴菲特投资相关核心术语定义。

| 术语 | 英文 | 简要定义 |
|---|---|---|
| 内在价值 | Intrinsic Value | 企业未来现金流的折现值 |
| 护城河 | Moat | 企业的持久竞争优势 |
| 安全边际 | Margin of Safety | 买入价格与内在价值之间的差距 |
| 能力圈 | Circle of Competence | 投资者真正理解的业务范围 |
| 浮存金 | Float | 保险公司持有但不属于自己的资金 |
| 复利 | Compound Interest | 利息产生利息的增长效应 |
| 资本配置 | Capital Allocation | 管理层分配公司资金的决策 |
| 账面价值 | Book Value | 公司资产减去负债的净值 |
| 商誉 | Goodwill | 收购价格超过净资产的部分 |
| 机会成本 | Opportunity Cost | 选择一项投资而放弃其他选择的代价 |
| 所有者盈余 | Owner Earnings | 巴菲特定义的真实盈利能力指标 |
| 特许经营权 | Franchise | 具有定价权和持久需求的业务 |
| 套利 | Arbitrage | 利用价格差异获取无风险利润 |
| 控制类投资 | Control Investments | 获得公司控制权的投资 |
| 私有化交易 | Private Transaction | 以私人协商价格进行的交易 |
